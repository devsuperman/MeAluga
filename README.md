## BackEnd

### Enter the folder
```
cd /BackEnd
```

### Run
```
dotnet run
```

## FrontEnd

### Enter the folder
```
cd /FrontEnd
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
